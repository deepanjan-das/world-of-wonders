Photo credits

The World of Wonders website is photo heavy. I have showcased beautiful photos of World Heritage sites. These photos are obtained through API calls to Flickr and 500px.

However, in some occasions, I had to use photos and graphics apart from these sources, without any API calls. This page is dedicated to crediting those photos and graphics used in my website.

Photos & graphics on home page

The photos of some World Heritage sites are used in the auto fading slideshow on the home page. The reason for putting them on my website folder was so that it would be easy to crop them into the desired dimensions and resize them as necessary. All these photos are from 500px.
